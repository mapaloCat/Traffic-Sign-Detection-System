# Traffic-Sign-Detection-System

This repository is about a Computer Vision project that aims at building a traffic sign detection algorithm. The â€œdetector" CNN determines whether a region proposal is a traffic sign or background noise and it classifies it to the corresponding class. In the currect implementation, the background noise is considered an extra class to the 43 classes of the ground truth traffic signs. The presented solution achieves 84.5% mAP (mean Average Precision) in a test set composed of 300 images. The execution time is 738 seconds.
